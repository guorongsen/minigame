# 隐私政策模板（示例）

> 请务必由你方业务/法务审核后再发布，以下仅为项目模板。

## 1. 我们收集的信息
- 设备基础信息（用于运行与性能适配）
- 游戏存档信息（用于保存游戏进度）
- 崩溃与异常日志（用于问题定位）

## 2. 信息使用目的
- 提供与维护游戏功能
- 改善稳定性与性能体验
- 统计分析游戏功能使用情况

## 3. 信息共享与披露
- 原则上不向第三方出售个人信息
- 如接入第三方服务（例如广告平台），将按其合规要求披露

## 4. 信息存储与安全
- 采取合理技术措施保护数据安全
- 存储期限以实现产品功能所需为限

## 5. 未成年人保护
- 未成年人应在监护人指导下使用产品
- 如涉及个人信息处理，遵循相关法律法规要求

## 6. 用户权利
- 用户可申请查询、删除相关信息（按平台能力与法规执行）
