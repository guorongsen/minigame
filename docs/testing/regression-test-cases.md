# 发布前回归用例

## A. 基础流程
1. 首次进入游戏 -> 开始一局 -> 正常结算 -> 返回首页
2. 切后台再回前台，游戏继续可玩
3. 暂停/继续/退出流程正常

## B. 战斗与成长
1. 升级三选一正常出现并可生效
2. 宝箱出现、拾取、变异面板流程正常
3. 进化触发后效果正确
4. 人物融合触发后属性生效

## C. 章节系统
1. 第 1 章可开局，后续章节按通关逐步解锁
2. 未解锁章节不能开局并有提示
3. 通关后星级计算正确（通关/无复活/击杀目标）
4. 首通奖励只发一次，且会解锁下一章

## D. 经济与存档
1. DNA 结算正确，双倍奖励逻辑正确
2. 武器碎片结算正确，每日宝箱奖励正确
3. 武器升级消耗和技能解锁正确
4. 重启游戏后进度完整保留

## E. 广告流程
1. 各广告位可触发/可拦截（冷却、门槛）逻辑正确
2. 广告成功给奖励，失败不给奖励
3. 广告不可用时提示正确且不阻塞流程

## F. 每日系统
1. 每日任务刷新与领取正确
2. 每日挑战奖励每日仅一次
3. 每日宝箱每日仅一次
