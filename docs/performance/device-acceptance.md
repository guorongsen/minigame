# 真机性能验收矩阵

## 目标基线
- 战斗常态：操作流畅，无明显输入延迟。
- 高压场景：允许短时掉帧，但不允许连续卡死/明显停顿。
- 整局稳定：无黑屏、无崩溃、无异常退出。

## 设备分层建议
- 低端机：2~3 年前主流安卓机（重点）
- 中端机：近 1~2 年安卓机
- 高端机：近 1 年旗舰安卓或 iOS

## 场景用例（每台至少跑 2 局）
1. 标准模式前中期（第 1~5 章）
2. 标准模式后期（第 12+ 章）
3. 无尽模式高压（至少生存到多批首领）
4. 每日挑战（含目标完成结算）
5. 广告相关流程（若可用）

## 记录表（建议）
- 机型 / 系统版本
- 模式 / 章节
- 性能档位（高画质/均衡/性能）
- 是否出现明显卡顿（Y/N）
- 是否出现黑屏/崩溃（Y/N）
- 备注（复现步骤）

## 验收结论
- 低端机可接受：`性能模式` 下可完整游玩且无致命问题
- 中高端机可接受：`均衡模式` 下体验稳定
